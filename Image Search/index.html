How to use:


go to https://ucshadow-img-search.herokuapp.com/search/ + your search term


example: https://ucshadow-img-search.herokuapp.com/search/Witcher3


you can add an offset to the query to specify the start point (in the result array) of the response


example: https://ucshadow-img-search.herokuapp.com/search/Witcher3?offset=10


also you can view the latest 10 queries at https://ucshadow-img-search.herokuapp.com/latest
